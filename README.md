
## Laravel Vue CRUD with Login

 Laravel Vue CRUD with Login is intended for beginner web developer who wants to learn Laravel Vue.

## How to use

- composer install.
- npm install.
- configure database(env)
- run laravel migration
- run laravel database seeder
